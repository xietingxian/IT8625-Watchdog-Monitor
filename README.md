# IT8625 Watchdog Monitor (工控机硬件监控与看门狗系统)

## 📖 项目简介 (Introduction)
本项目是一个基于 **WinForms (.NET Framework)** 开发的工控机硬件综合管理工具。
专为搭载 **ITE IT8625** 超级IO芯片（SuperIO）的工控主板设计，集成了硬件状态监控、看门狗（Watchdog）自动复位保护以及系统日志记录功能。

该程序目前已应用于产线及客户现场，用于保障工控设备在无人值守环境下的 7x24 小时稳定运行。

## 🚀 核心功能 (Features)

### 1. 硬件状态实时监控
*   **多维度传感器读取：** 通过底层接口直接读取 IT8625 芯片数据。
*   **实时数据显示：**
    *   **主控芯片型号：** 自动识别并显示 SuperIO 型号（IT8625）。
    *   **温度监控：** 实时显示 CPU 温度、主板环境温度及硬盘温度。
    *   **风扇转速：** 实时监测 CPU 风扇转速（RPM），异常停转预警。

### 2. 高可靠性看门狗 (Watchdog)
*   **自动喂狗机制：** 内置定时器，按设定周期向底层寄存器发送喂狗信号，防止系统死机。
*   **系统保护：** 当应用程序崩溃或系统无响应时，硬件看门狗将强制复位工控机，实现无人值守自恢复。

### 3. 自动化运维功能
*   **开机自启动：** 通过操作 Windows **注册表 (Registry)** 的 `Run` 键值，实现软件随系统启动并最小化至托盘。
*   **数据日志记录：** 
    *   集成文件 I/O 模块，每隔指定时间（如10分钟）自动将当前系统状态（温度、喂狗次数、运行时间）写入 `.txt` 日志文件。
    *   便于售后人员追踪设备运行历史与故障排查。

## 🛠️ 技术栈 (Tech Stack)
*   **开发框架：** Windows Forms (WinForms) / C#
*   **底层驱动：** WinRing0  (用于访问 SuperIO 寄存器)
*   **系统交互：** Windows Registry API (注册表管理), System.IO (文件流处理)
*   **硬件协议：** SMBus
